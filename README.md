# cTAKES-mCODE
This repository is an extension of the Apache cTAKES toolkit, tailored towards extracting mCODE (minimal Common Oncology Data Elements) from clinical text.
## Prerequisites
- All prerequisites for [Apache cTAKES 6.0.0](https://ctakes.apache.org/) must be met.
- Release file for [Apache cTAKES 6.0.0](https://github.com/apache/ctakes/releases/download/ctakes-6.0.0/apache-ctakes-6.0.0-bin.zip) must be downloaded and extracted to a local directory.
